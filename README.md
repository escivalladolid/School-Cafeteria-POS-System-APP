# School-Cafeteria-POS-System-APP

A mobile Point of Sale (POS) application developed for a school cafeteria.
This system was created as a school project to manage cafeteria transactions and ordering processes.

## Features
- Mobile-based POS system
- Food ordering system
- Product/menu management
- Order processing
- Database integration
- Transaction records

## Technologies Used
- Expo Go
- React Native
- PHP
- MySQL
- XAMPP
